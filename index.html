NSM-RPMs

NSM RPMs for Enterprise Linux and Fedora

Summary

NSM RPMs is a set of RPM packages for Enterprise Linux (RedHat and derivatives such as CentOS) and Fedora Linux related to Network Security Monitoring.

The following distributions are currently supported:

  • EL 6/i386 (RedHat EL, CentOS, Scientific Linux)
  • EL 6/x86_64 (RedHat EL, CentOS, Scientific Linux)
  • Fedora 17/i386
  • Fedora 17/x86_64

Getting Started: EL6

  1. Install EPEL, as it is required on EL6: 
    
rpm -Uvh http://nsm-rpms.unx.ca/nsm-rpms/el/6/epel-release-6-7.noarch.rpm
  2. Install the NSM RPMs repository configuration package: 
    
rpm -Uvh http://nsm-rpms.unx.ca/nsm-rpms/el/6/x86_64/nsm-release-el-6-1.el6.noarch.rpm

Getting Started: Fedora 17


rpm -Uvh http://nsm-rpms.unx.ca/nsm-rpms/fedora/17/x86_64/nsm-release-fc-17-1.fc17.noarch.rpm

Installing Suricata

To install the latest version of Suricata: 


sudo yum install nsm-suricata-latest

Installing Snort

To install the latest version of Snort: 


sudo yum install nsm-snort-latest

Listing All Packages


yum list 'nsm-*'

Direct Package Downloads

Packages can be downloaded directly from http://nsm-rpms.unx.ca/.
